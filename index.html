Album Tags

The Album Tags app is one of the largest projects I've worked on to date. I built it in Node.js using Express and Bootstrap 4 with ejs for the view engine. The app makes calls to two Apple Music API's and displays details about different records. In addition, it queries a MongoDB database of custom user tags and displays the specific tags pertaining to an album.

This functionality combines to allow users to find records they like, review the tags used to describe them, then use the tags to search for similar records. I used Firebase authentication to limit who has the ability to add tags to albums, but any user can access the site and use the tag functionality to find some cool music.

The repository linked below has detailed documentation about the process of creating the app, so if you are curious to learn more I suggest starting there, or taking a look through the live app which is currently hosted using Heroku.
